# PRODUCT.md — IFA-Report 產品定義

產品名稱：IFA Financial Report（IFA 財務規劃報告工具）
目標市場：台灣，繁體中文 UI
使用者：獨立理財顧問（IFA），協助客戶建立財務檔案、視覺化呈現財務狀況，並產出可交付的報告

## 核心問題（JTBD）

IFA 需要一個工具，讓他們能快速建立客戶財務檔案、視覺化呈現財務狀況，並將互動報告分享給客戶查看。

## 功能範圍（已實作）

| 功能 | 說明 |
|------|------|
| 多客戶管理 | 新增、切換、刪除客戶；localStorage 持久化；向後相容 schema migration（目前 v10） |
| 現金流分析 | 收入/支出圓餅圖 + 瀑布圖，顯示月淨現金流與健康指標 |
| 資產成長投影 | 三情境（保守／基準／積極）+ 月定期投入折線圖 |
| 退休規劃 | 缺口分析、必要月儲蓄、30 年提款模擬 |
| 資產配置分析 | 類別分布圓餅、幣別分布、用途分布；快照比對（前期 vs 本期） |
| 資產快照 | 記錄特定時間點資產狀態，供期間報酬比較 |
| 多幣別支援 | TWD/USD/JPY/EUR/GBP/HKD/USDT；即時 FX 匯率（open.er-api.com），可手動覆蓋 |
| 報告分享 | IFA 建立分享連結（密碼保護），客戶端以唯讀方式查看互動圖表 |
| PDF 匯出 | html2canvas 截圖 `.report-page` → jsPDF 多頁整合 |
| 設計系統 | Analytics Dashboard（Light Professional）風格；IBM Plex Sans；深藍 #1E40AF + 琥珀 #D97706 |

## 使用者輸入資料（InputForm 6 tabs）

1. **基本** — 出生年份、職業、諮詢重點、諮詢建議（可多筆）
2. **收支** — 收入項目（類型/頻率/成長率/備註）、支出項目（類別/頻率/備註）、通膨預估率
3. **資產** — 資產項目（類別/幣別/機構/用途）、負債項目（長期/流動）
4. **投資** — 風險承受度（保守/穩健/積極）、自訂報酬率、每月定期投入、目標配置、容許偏離
5. **支出** — 重大支出計劃（項目/金額/年份）
6. **退休** — 目標退休年齡、退休餘命、目標月退休現金流

## 客戶端 View（唯讀，分享連結）

- `src/pages/ClientView/` — 密碼驗證後進入
- `BasicInfoPage.tsx` — 唯讀展示所有客戶輸入資料
- `ChartsPage.tsx` — 依 `visible_modules` 顯示開放的互動圖表
- `PasswordGate.tsx` — 密碼驗證元件

## UX 原則

- 繁體中文介面
- 大數字使用「萬/億」顯示（`fmtNTD` / `fmtAmount` 函數）
- `.report-page` class 標記截圖範圍，不含 sidebar/導覽列
- 報告幣別可切換（TWD/USD/JPY 等），換算以 FX 匯率為準
- 無需登入（IFA 端），降低使用門檻

## 功能邊界（不在範圍）

- 無 IFA 帳號系統（客戶分享用 Supabase，IFA 本地無帳號）
- 無多人協作
- 無行動裝置優先設計（桌面為主）
- 無多語系（繁體中文唯一）
