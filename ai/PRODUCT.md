# PRODUCT.md — IFA-Report 產品定義

產品名稱：IFA Financial Report（IFA 財務規劃報告工具）
目標市場：台灣，繁體中文 UI
使用者：獨立理財顧問（IFA），協助客戶建立財務規劃與退休分析

## 核心問題（JTBD）

IFA 需要一個工具，讓他們能快速建立客戶財務檔案、視覺化呈現財務狀況，並產出可直接交付給客戶的 PDF 報告。

## 功能範圍（已實作）

| 功能 | 說明 |
|------|------|
| 多客戶管理 | 新增、切換、刪除客戶；localStorage 持久化；向後相容 schema migration |
| 現金流分析 | 收入 pie chart + 支出 waterfall chart，顯示月淨現金流 |
| 資產成長投影 | 三情境（保守／基準／積極）+ 月定期投入折線圖 |
| 退休規劃 | 缺口分析、必要月儲蓄、30 年提款模擬 |
| PDF 匯出 | html2canvas 截圖 .report-page → jsPDF 多頁整合 |

## 使用者輸入資料（InputForm 5 tabs）

1. **收入**：薪資、其他收入（可多筆，含備註）
2. **支出**：固定支出、變動支出
3. **資產**：現金、不動產、保險、股票、基金、定存、債券、加密貨幣、其他
4. **負債**：長期負債、流動負債
5. **目標**：退休年齡、退休目標月收入、重大支出事件

## UX 原則

- 繁體中文介面，貨幣格式 TWD（Intl.NumberFormat）
- 大數字使用「萬/億」顯示（fmtNTD 函數處理）
- `.report-page` class 標記截圖範圍，必須乾淨（不含 sidebar/導覽列）
- 無需登入，無後端，降低 IFA 使用門檻

## 功能邊界（不在範圍）

- 無帳號系統
- 無雲端同步
- 無多人協作
- 無外部 API 串接
- 無行動裝置優先設計（桌面為主）
